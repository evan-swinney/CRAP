The Common Rocketry Avionics Package (CRAP) project is a rocket avionics package, designed specifically for UAH Space Hardware Club to be flown on the multistage team's rocket.  This payload will also serve as an in-house alternative to L1 Certifications and general rocketry for data collection and GPS tracking.  This payload will fire all rocket ordance and motors as well as provide data telemetry and storage.
